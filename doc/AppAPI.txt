App发送验证码:
	POST
		/app/user/sendRegVerifyCode?phoneNumber=xxxxxxxxx
	返回值:
		{
			"state": 0,
			"msg": "xxxxxxx"
		}

App用户登录:
	POST
		/app/user/login?userName=xxxx&password=xxxxxxx
		或
		/app/user/login?phoneNumber=xxxx&password=xxxxxxx
	返回值:
		{
			"state": 0,
			"msg": null,
			"sessionId": "xxxxxxxxxxxxxxxxxxxxx",
			"userInfo": {
				"phone": "xxxxxxxx",
				"userName": "xxxxxxxx",
				"email": "xxxxxxxxx"
			}
		}

App用户注册:
	POST
		/app/user/register?phoneNumber=xxxxxxx&userName=xxxxxxxx&password=xxxxxx&verifyCode=xxxxxxx&email=xxxxxxxxxx
	返回值:
		{
			"state": 0,
			"msg": "xxxxxxx"
		}

上传图片：
	POST
		/app/image
		content-type:  multipart/form-data
	参数：sessionId   登录返回的sessionId
		  image         图片文件内容
	返回值:   {
				"imageId": 6,
				"width": 6000,   //原图尺寸
				"height": 2315,   //原图尺寸
				"url": "http://localhost:6060/assets/user/1/D8F219B9D0C0470C961074650E79D3DC/e2fee30d51cc481ab8a4e7f3419014b7.jpg",   //图片url
				"errcode": 0,
				"errmsg": null
			 }
删除上传的图片:
	DELETE
		/app/image?imageId=xxx&sessionId=xxxx
	返回值:
			{
				"errcode" : 0,
				"errmsg" : null
			}


请求重置密码:  (发送重置密码验证码到手机)
    POST
        /app/user/requestResetPassword
    参数: phone: 手机号码
	返回值:
			{
				"state" : 0,
				"msg" : null
			}

重置密码:
    POST
        /app/user/resetPassword
    参数: phoneNumber: 手机号码  或  userName: 用户名
         verifyCode: 上一个请求中发到用户手机上的验证码
         newpassword: 新密码
	返回值:
			{
				"errcode" : 0,
				"errmsg" : null
			}

使用条款:

    返回值:
        content-type: text/html

常见问题:

检查app版本 (android)
    GET
        /assets/app/client.json
    返回值:
        {
            "version":"0719",
            "fileName":"JoySpace_client-0719.apk"
        }
    apk下载地址:
        /assets/app/<fileName>


推荐自助机
    GET
        /app/printStation/findByDistance
    参数:
        longitude:  位置精度(double)
        latitude: 位置纬度(double)
        radius:   搜索半径(int, 单位米)

    返回值:
        {
				"state" : 0,
				"msg" : null,
				"result" : [
				    { "id" : xx, "address": "xxxx", "longitude" : xxxx.xxx, "latitude": xxxx.xxx },
                    { "id" : xx, "address": "xxxx", "longitude" : xxxx.xxx, "latitude": xxxx.xxx },
				    ...
				    { "id" : xx, "address": "xxxx", "longitude" : xxxx.xxx, "latitude": xxxx.xxx }
				]
        }
